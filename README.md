 StudentBudget Tracker

StudentBudget Tracker is a full-stack web application designed to help students manage their personal finances efficiently.  
The application allows users to track income and expenses, view their total balance, and develop better budgeting habits.  
It includes a frontend interface and a Node.js/Express backend with database integration.

Features
- User registration and login
- Add, update, and delete income and expenses
- Track total balance
- Secure backend with authentication
- Responsive and user-friendly interface

Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Database: MySQL
- Version Control: Git & GitHub
- Deployment: Render

How to Run the Project Locally
1. Clone the repository  
   `git clone https://github.com/username/studentbudget-tracker.git`

2. Navigate to the project folder  
   `cd studentbudget-tracker`

3. Install dependencies  
   `npm install`

4. Start the server  
   `npm start`

5. Open your browser and visit  
   `http://localhost:3000`
